# pokedex_flutter

Projeto Flutter de uma Pokedex com:

- carregamento paginado da PokeAPI;
- filtro por nome ou numero;
- cards com cor dominante extraida da imagem;
- gerenciamento de estado com MobX.

## Documentacao

Para estudar o projeto em arquivos separados, leia:

- [docs/GUIA_DE_ESTUDO.md](docs/GUIA_DE_ESTUDO.md)

## Comandos uteis

```bash
flutter pub get
flutter run
dart run build_runner build
```
