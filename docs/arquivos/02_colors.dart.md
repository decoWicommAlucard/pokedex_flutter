# colors.dart

## O que esse arquivo faz

`lib/colors.dart` centraliza a cor principal usada na interface.

## Codigo

```dart
const primaryColor = Color(0xFF2E3156);
```

## Por que isso existe

Em vez de repetir a mesma cor em varios lugares, o projeto guarda essa cor em um unico arquivo.

## Vantagem pratica

Se voce quiser trocar a identidade visual depois, basta mudar este valor.

## Onde essa cor e usada

- titulo da tela inicial;
- subtitulo da tela inicial;
- icone do campo de busca;
- textos dentro dos cards.
